La primera versión del pryecto está Proyecto_FicComp.ipynb, 
donde se construye el hamiltoniano con el sistema acomplado, 
y se aplica Runge-Kutta para encontrar las solución.

El archivo correcto con la versión del proyecto final es Proyecto_FicComp-Final3.ipynb, 
y se hace una simulación para diferentes 'k' del resorte que une las partículas. 
Adicionalmente se implementó una animación para ver la simulación de las partícula acopladas.
